# Введение

Калькулятор индекса массы тела (BMI Calculator) — это приложение,
предназначенное для расчёта индекса массы тела пользователя на основе
его роста и веса.

Программа позволяет определить, находится ли вес пользователя в пределах
нормы, недостаточен ли он или превышает рекомендованные значения.

Основные возможности программы:

- ввод роста и веса пользователя
- автоматический расчёт индекса массы тела
- отображение категории веса
- вывод рекомендаций

# Примеры использования

1. При вводе значения роста пользователь может указать свой рост в сантиметрах.
2. При вводе значения веса пользователь вводит свой вес в килограммах.
3. При нажатии на кнопку **«Рассчитать»** программа вычисляет индекс массы тела.
4. После расчёта на экране отображается результат BMI.
5. Программа определяет категорию веса пользователя (недостаточный, нормальный, избыточный).
6. При нажатии кнопки **«Очистить»** все поля ввода очищаются.
7. При вводе некорректных данных программа выводит сообщение об ошибке.
8. После расчёта пользователь может сохранить результат.
9. Программа может отображать рекомендации по поддержанию нормального веса.
10. Пользователь может повторно выполнить расчёт, введя новые данные.
